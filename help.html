How to use this?

For Teacher/Mentor

  • Make sure that you have logged in correctly in the Admin Section.
  • Once done it, a quiz code will be generated .
  • Go to the Build Quiz section and enter the same quiz code you get on log in.
  • After entering the code a standard template will appear numberwise ,In this you can generate quiz upto 5 questions.
  • Enter the question and its options and also the right answer as present in any option, Note: The content in correct option and the right answer you are entering must be same
  • Once fill the first part of question, click on the Save button to save this data and then click on the Next button to continue the process
  • Failing to do so, will lead to cause error and your data for a particular question will not be stored
  • After entering all the questions click on the Generate button ,your quiz will be generated
  • Share quiz code with the students to give this quiz.

For Students

  • Make sure that you have logged in correctly in the Student Section.
  • Once done it, Go to the Start Quiz section and enter the quiz code given by your teacher or host
  • On clicking Start button your quiz will get started and you will get 90 seconds to attempt.
  • Select appropiate answer and click on the Next button.
  • After giving all the 5 questions click on Submit button
    your quiz will be submitted and you can all see your score in Your Score section.
  • If you failed to complete the quiz on given time, it will be submitted automatically.
